# Hermes Suno Music Agent

A comprehensive AI agent for creating complete music video projects using Suno and various image/video generation tools. Built specifically for the Hermes agent framework.

This agent guides you through a structured, high-quality workflow from song concept to final deliverables — including lyrics, image prompts, video prompts, and YouTube metadata.

## Features

- **8-Step Interactive Workflow** — Strict user-controlled pacing (you decide when to move to the next stage)
- Full song package generation (lyrics, image prompts, video prompts)
- YouTube-optimized title, description, and keyword generation
- Comprehensive prompting guides for Suno, image tools, and video tools
- Music production and songwriting references included
- Generic setup — works with any music style or channel

## Installation

1. Copy the `Hermes-Suno-Music-Agent` folder into your Hermes skills directory:
   ```
   ~/.hermes/skills/Hermes-Suno-Music-Agent/
   ```

2. Restart Hermes or reload skills.

## How to Use

When you first activate the agent, it will run an initialization process:

1. Ask for your music projects root folder
2. Ask for your channel/project name
3. Ask for your primary music style and vibe
4. Optionally create the basic project structure

After setup, the agent follows the structured 8-step creative workflow.

## Workflow

1. Title Generation
2. Genre / Style Definition
3. Lyrics & Suno Prompt
4. Video Vision
5. Image Breakdown
6. Image Prompt Generation
7. Video Prompt Generation
8. YouTube Metadata

The agent will wait for your confirmation before moving to each next step.

## Philosophy

This agent prioritizes quality and intentional creation over speed. It is designed for creators who want well-crafted, coherent music video projects.

## Requirements

- Hermes agent framework
- Access to Suno (and optionally image/video tools such as Flux, Veo 3, Kling, etc.)
