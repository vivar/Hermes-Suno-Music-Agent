# Suno Tags Reference

Comprehensive list of effective Suno tags for high-quality music generation. Use these when crafting style prompts.

## Structure Tags

- [Intro] — Instrumental opening section
- [Verse] — Main narrative/storytelling sections
- [Pre-Chorus] — Build-up before the chorus
- [Chorus] — Main hook, most repeated section
- [Post-Chorus] — Hook extension or additional catchy part
- [Bridge] — Contrasting section, often emotional or instrumental
- [Outro] — Final section, usually fades or resolves

## Best Practices

- Place the most important tags in the first 20–30 words of the style prompt.
- Limit to 2–3 instruments and 1–2 genres to avoid confusion.
- Combine genre + mood + instrument tags for best results.
- Use specific instrument tags rather than generic ones when possible.
- Use [Weirdness] and [Style Influence] parameters to control how closely Suno follows the prompt.
