# KlingAI Video Reference

## Overview
KlingAI is a high-quality AI video generation platform known for strong motion quality, character consistency, and flexible creative control. It performs well with both realistic and stylized content and offers good tools for extending clips and maintaining subject coherence.

## Prompt Structure
- Clear subject + action + environment description
- Camera and motion instructions
- Style and mood qualifiers
- Duration and aspect ratio notes when relevant

**Example Prompt:**
"A female singer performs on a rooftop at night with city lights in the background. She moves expressively while singing. Camera slowly orbits around her. Cinematic lighting with cool blue and warm orange accents. Smooth, emotional motion with natural fabric flow."

## Best Practices
- Use detailed subject descriptions for better character consistency
- Specify camera paths explicitly (orbit, push-in, crane up)
- Add environmental interactions (wind, light changes, reflections)
- Generate in segments and use extension features for longer sequences
- Leverage reference image uploads for consistent characters across clips

## Strengths
- Strong character and face consistency across generations
- Good motion naturalness and physics
- Flexible aspect ratios and duration options
- Solid performance on both realistic and artistic styles
- Useful tools for motion brush and regional control

## Things to Avoid
- Sudden drastic style shifts mid-clip
- Overly crowded group scenes without clear focus
- Very rapid or erratic camera movements

## Advanced Tips
- Build a consistent character by generating a "hero image" first, then using it as reference
- For music videos: create performance shots + B-roll + abstract visualizers as separate clips
- Sync motion peaks with musical drops using timing notes from arrangement.md
- Use KlingAI's extension and loop features for ambient or repeating visual motifs
- Combine with generic-image.md principles when preparing input frames
